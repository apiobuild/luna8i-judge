Image Captioning Model Evaluation Report

Benchmarks three candidate vision models (ollama/llava, ollama/minicpm-v, qwen/qwen3.6-35b-a3b) on image captioning using 50 images from the VizWiz-Captions dataset, including 10 poor-quality unanswerable images. Uses gemini/gemini-3.1-flash-lite as the SOTA model and LLM-as-judge to score each candidate on faithfulness, completeness, conciseness, and instruction following.
